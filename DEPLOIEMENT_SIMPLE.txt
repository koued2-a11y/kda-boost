📌 ÉTAPE 1 : Créer un compte Vercel (1 minute)

   1. Allez sur https://vercel.com
   2. Cliquez "Sign Up"
   3. Choisissez "Continue with GitHub" (ou Email)
   4. Autorisez Vercel

   ✅ FAIT !


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━


📌 ÉTAPE 2 : Créer un repo GitHub (3 minutes)

   1. Allez sur https://github.com/new
   2. Nom du repo : kda-boost
   3. Description : Mon site de boost TikTok
   4. Privé ou Public (votre choix)
   5. Cliquez "Create repository"

   6. Dans votre PowerShell (dans le dossier kda-boost-app) :

      git init
      git add .
      git commit -m "KDA-Boost initial"
      git remote add origin https://github.com/VOTRE_USERNAME/kda-boost.git
      git branch -M main
      git push -u origin main

   ✅ FAIT ! Votre code est sur GitHub !


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━


📌 ÉTAPE 3 : Importer sur Vercel (2 minutes)

   1. Sur Vercel Dashboard, cliquez "New Project"
   2. Cliquez "Import Git Repository"
   3. Autorisez l'accès à votre repo GitHub
   4. Sélectionnez "kda-boost"
   5. Cliquez "Import"

   ✅ FAIT ! Vercel prépare votre projet !

     → Vérifiez que node_modules est dans .gitignore
     → Relancez le déploiement

   • Variables d'environnement manquantes ?
     → Project Settings → Environment Variables
     → Ajoutez-les et redéployez

   • Site ne fonctionne pas ?
     → Vérifiez que vous avez exécuté le script SQL dans Supabase
