❌ PROBLÈME : Impossible de se connecter à votre base de données Supabase


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ SOLUTION RAPIDE (5 MINUTES)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━


ÉTAPE 1 : Obtenez votre URL de connexion COMPLÈTE

   1. Allez sur https://supabase.com/dashboard
   2. Connectez-vous
   3. Sélectionnez votre projet
   4. Settings → Database
   5. Descendez à "Connection string"
   6. Sélectionnez "URI" ou "Session mode"
   7. Copiez TOUTE l'URL qui ressemble à :
   
      postgresql://postgres:[VOTRE-MOT-DE-PASSE]@db.ehmdqqmkzkstoerhenqb...


ÉTAPE 2 : Mettez à jour le fichier .env

   1. Ouvrez le fichier .env (dans ce dossier)
   2. Trouvez la ligne qui commence par DIRECT_URL=
   3. Remplacez TOUTE la ligne par :
   
      DIRECT_URL="[COLLEZ L'URL COMPLÈTE D'ÉTAPE 1]"
   
   4. SAUVEGARDEZ le fichier


ÉTAPE 3 : Testez la connexion

   Double-cliquez sur : TESTER_CONNEXION.bat
   
   Si ✅ : Continuez à l'étape 4
   Si ❌ : Vérifiez l'étape 1 et 2


ÉTAPE 4 : Relancez l'installation

   Double-cliquez sur : INSTALLER_PUIS_LANCER.bat


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
