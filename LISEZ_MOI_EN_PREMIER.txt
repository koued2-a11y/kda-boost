1  npm install
2  Configurer .env (copiez .env.example)
3  npx prisma generate
4  npx prisma db push
5  npm run dev

 Ouvrez http://localhost:3000

 STACK TECHNOLOGIQUE

Frontend  : Next.js 14 + TypeScript + Tailwind CSS

Auth      : NextAuth.js
Animations: Framer Motion
Icons     : React Icons



 ROUTES DISPONIBLES
